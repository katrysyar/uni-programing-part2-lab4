# Лабораторна робота №4

_**Виконала: Рябцева Катерина, ІО-з51**_

**Тема**: Класи в мові програмування Java

**Мета**: Ознайомлення з класами. Використання існуючих та створення власних класів в мові Java.

**Завдання**: 
Створити клас із виконавчим методом, в якому створити масив з об’єктів класу визначеному варіантом (п. 2). Відсортувати масив, за одним з полів за зростанням, а за іншим — за спаданням2, використовуючи при цьому стандартні засоби сортування (). Після цього знайти в масиві об’єкт, який ідентичний заданому. Всі змінні повинні бути описані та значення їх задані у виконавчому методі. Код повинен бути детально задокументований.

**Отриманий індивідуальний варіант**: 
  * варіант завдання: Визначити клас навчальний заклад, який складається як мінімум з 5-и полів;

**Висновок**: У ході виконання лабораторної роботи було здобуто практичні навички створення та використання власних класів у мові Java. Було розроблено клас "Навчальний заклад" із необхідними полями та реалізовано роботу з масивом об'єктів цього класу. Успішно застосовано стандартні засоби Java (Arrays.sort та Comparator) для багатокритеріального сортування масиву (за зростанням одного поля та спаданням іншого). Також було закріплено розуміння механізмів порівняння об'єктів за їхнім вмістом (перевизначення методу equals) для пошуку заданого елемента та набуто базового досвіду документування коду.
